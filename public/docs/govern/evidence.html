Run & Govern

Evidence DAG.

An evidence DAG is the directed acyclic graph that records every fact about a deployed artifact: what captures fed it, which teachers labelled those captures, which evals it passed, what its k-score history looks like, what drift signals have fired against it, and every rollout decision it has been part of. The DAG is queryable, exportable, and signed.

Node types

NodeWhat it represents
captureA request · response pair recorded by the gateway or imported via kolm import.
labelA teacher's output on a capture, with the teacher id and the cost.
distill_runA training run that consumed labels and produced an artifact candidate.
evalA held-out evaluation result — pass/fail with the score and the rubric id.
artifactA signed .kolm produced by a successful distill_run that cleared its evals.
rolloutA deployment event — which namespace, what split, when.
drift_signalA drift detector fire bound to the artifact and the namespace.
assurance_claimA claim in the artifact's assurance case backed by a subgraph of evidence.

Edges

Edges are typed and immutable. label DERIVED_FROM capture, distill_run CONSUMED label, artifact PRODUCED_BY distill_run, eval EVALUATED artifact, rollout DEPLOYED artifact, drift_signal OBSERVED_AGAINST artifact, assurance_claim SUPPORTED_BY {eval | distill_run | capture}. The set is closed; new edges land as new releases not patches.

Querying the graph

# Walk back from a deployed artifact to every capture it was trained on:
kolm evidence trace art_a1b2c3d4 --upstream capture

# Show every drift signal that has ever fired against this artifact:
kolm evidence trace art_a1b2c3d4 --node drift_signal

# Export the assurance case for art_a1b2c3d4 as a single JSON-LD blob:
kolm evidence export art_a1b2c3d4 --format jsonld > assurance.jsonld

Signing

The DAG root for an artifact is signed at kolm compile time and re-anchored on every drift_signal or rollout edge addition. The signature chain is verifiable independently of the kolm control plane:

kolm evidence verify art_a1b2c3d4
# DAG root: bafyreif7q...
# signed:   ks_4b7bc3b1... at 2026-05-12T04:11Z
# chain:    OK through 318 edges
