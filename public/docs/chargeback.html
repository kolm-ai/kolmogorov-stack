Chargeback and department attribution.

Roll per-namespace billing-breakdown rows up to departments. For finance teams that need a single line item per cost center instead of a sprawl of namespace names.

What it is

Chargeback layers on top of the W465 billing-breakdown rows (per-namespace cost attribution sourced from the event-store ledger). Each namespace carries an optional department tag; chargeback groups the namespaces by that tag and emits a roll-up row per department per period.

Department vocab

Eight canonical departments:

  • engineering
  • product
  • ops
  • sales
  • marketing
  • support
  • research
  • other -- catch-all for namespaces with no tag, or a tag outside the canonical set.

Free-form department strings are accepted at namespace creation time but normalized to the canonical set at roll-up time. The unnormalized string is preserved in the raw row.

Period bounds

Period is a calendar month in UTC, formatted YYYY-MM. The bounds are [YYYY-MM-01T00:00:00Z, YYYY-MM+1-01T00:00:00Z). The API rejects malformed months (e.g. 2026-13) with HTTP 400 invalid_period; it does not silently roll month 13 into January of the next year.

Group-by

Three group-by axes:

  • department -- the standard roll-up.
  • namespace -- equivalent to kolm billing breakdown --by namespace; useful when you want chargeback in a CSV alongside namespace rows.
  • provider -- groups by upstream teacher provider (anthropic, openai, kolm-distill-teacher, vllm, etc.). Useful when finance asks "what did we spend on Claude vs OpenAI this month."

CSV and JSON export

Both formats are first-class. CSV is RFC 4180; JSON is the same shape as the API envelope.

kolm chargeback --period 2026-05 --by department --format csv > chargeback-2026-05.csv
kolm chargeback --period 2026-05 --by department --format json --pretty

CSV columns: period, department, namespace_count, cost_micro_usd, captures, distill_runs, inference_calls.

CLI

kolm chargeback --period 2026-05 --by department
kolm chargeback --period 2026-05 --by provider --format csv
kolm chargeback --period 2026-04 --by namespace --json

API route

Auth-gated. See the dashboard at /account/chargeback.

GET /v1/chargeback?period=2026-05&by=department&format=json
