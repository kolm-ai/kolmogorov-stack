Connect Gemini

Google AI Studio ships an OpenAI-compatible surface at generativelanguage.googleapis.com/v1beta/openai. Point the kolm.ai proxy at it, point your OpenAI SDK at the proxy, and Gemini calls land in your local lake just like any other provider.

Setup

Set the proxy upstream to the Gemini OpenAI-compat URL. The API key is your AI Studio key, forwarded unchanged.

export OPENAI_BASE_URL=http://127.0.0.1:8787/v1
export OPENAI_API_KEY=AIza-your-gemini-key
export KOLM_FORWARD_TO=https://generativelanguage.googleapis.com/v1beta/openai
kolm connect start --upstream gemini

Model IDs

Use the same model ids you would send to AI Studio directly.

ModelModel id
Gemini 2.5 Flashgemini-2.5-flash
Gemini 2.5 Progemini-2.5-pro
Gemini 3.0 Flashgemini-3.0-flash
Gemini 3.0 Progemini-3.0-pro

Python SDK

from openai import OpenAI
client = OpenAI(base_url="http://127.0.0.1:8787/v1")

r = client.chat.completions.create(
 model="gemini-2.5-flash",
 messages=[{"role": "user", "content": "Three SQL window function gotchas."}],
)
print(r.choices[0].message.content)

TypeScript SDK

import OpenAI from "openai";
const client = new OpenAI({ baseURL: "http://127.0.0.1:8787/v1" });
const r = await client.chat.completions.create({
 model: "gemini-3.0-flash",
 messages: [{ role: "user", content: "Two ways to detect memory leaks in Go." }],
});
console.log(r.choices[0].message.content);

Vision and multimodal

The OpenAI-compat Gemini endpoint accepts the same image_url content blocks the OpenAI SDK uses. Base64 data URLs work. The captured row carries modality=multimodal on the event.

r = client.chat.completions.create(
 model="gemini-2.5-flash",
 messages=[{
 "role": "user",
 "content": [
 {"type": "text", "text": "What is in this image?"},
 {"type": "image_url", "image_url": {"url": "data:image/png;base64,iVBORw0..."}}
 ],
 }],
)

Streaming

The proxy forwards SSE chunks unchanged. Latency to first token is measured on the upstream side, written to the ttft_ms lake field.

const stream = await client.chat.completions.create({
 model: "gemini-3.0-flash",
 stream: true,
 messages: [{ role: "user", content: "Stream a definition of CAP theorem." }],
});
for await (const c of stream) process.stdout.write(c.choices[0]?.delta?.content || "");
