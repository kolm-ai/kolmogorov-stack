kolm redteam

Run adversarial prompt suites against an artifact to surface jailbreaks, prompt injection, and policy bypass.

Usage

kolm redteam <artifact.kolm> [--suite <slug>] [--out <path>] [--json]

Flags

FlagDefaultDescription
requiredArtifact under test.
--suite k-defaultk-default, owasp-llm-top10, strongreject, or a path to a custom JSONL.
--out stdoutWhere to write the per-prompt report.
--jsonoffJSON output.

Examples

kolm redteam ./out/recipe.kolm
kolm redteam ./out/recipe.kolm --suite owasp-llm-top10 --out ./redteam.json --json

Notes

Suites are versioned. Results are deterministic for a given (artifact, suite) pair so CI can pin them.
