kolm pubkey

Print, export, or rotate the local Ed25519 public key used by kolm attest.

Usage

kolm pubkey [--export <path>] [--rotate] [--json]

Flags

FlagDefaultDescription
--export stdoutWrite the public key to a file (PEM).
--rotateoffGenerate a fresh keypair (existing key archived).
--jsonoffJSON envelope.

Examples

kolm pubkey
kolm pubkey --export ./kolm.pub
kolm pubkey --rotate

Notes

Keys live under ~/.kolm/keys/. Rotation archives the previous key so old signatures still verify.
