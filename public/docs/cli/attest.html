kolm attest

Sign an artifact or document with the local Ed25519 keypair. Produces a sidecar .sig verifiable by kolm verify.

Usage

kolm attest <file> [--key <path>] [--out <path.sig>]

Flags

FlagDefaultDescription
requiredFile to sign.
--key ~/.kolm/keys/ed25519Private key path.
--out <file>.sigWhere to write the signature.

Examples

kolm attest ./out/recipe.kolm
kolm attest ./out/audit.cef --out ./audit.sig

Notes

Pair with kolm pubkey to publish your verification key. Sigstore integration is available via kolm sigstore-attest.
