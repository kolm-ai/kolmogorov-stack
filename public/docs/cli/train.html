kolm train

Run a training loop from captured data. Alias for kolm distill from-captures with sensible defaults.

Usage

kolm train [--teacher ] [--student ] [--since ] [--namespace ]

Flags

FlagDefaultDescription
--teacher autoTeacher model. Defaults to the highest-k-score available teacher.
--student autoStudent model. Defaults to the smallest student that fits the eval.
--since 7dCaptures window: 24h, 7d, 30d, or ISO range.
--namespace defaultCapture namespace to draw rows from.
--resume offResume an interrupted run.

Examples

kolm train
kolm train --teacher claude-sonnet-4-6 --student qwen2.5-3b-instruct
kolm train --namespace inbox --since 24h --resume auto

Notes

Writes checkpoints to ~/.kolm/runs/<run_id> with a row per step. Fail over to a backup teacher is on by default. See kolm distill for the lower-level surface.
