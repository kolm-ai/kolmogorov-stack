kolm vlm

Run a vision-language model (VLM) against an image or image set. CLIP / LLaVA / Qwen-VL backends.

Usage

kolm vlm <ask|caption|embed> <image.jpg|png> [--prompt <text>] [--model <slug>]

Flags

FlagDefaultDescription
ask-Ask a question grounded in the image.
caption-Generate a caption.
embed-Produce an embedding vector.
requiredImage file.
--prompt for askQuestion text.
--model defaultVLM backend.

Examples

kolm vlm caption ./photo.jpg
kolm vlm ask ./diagram.png --prompt "what does this show?"

Notes

Backed by workers/vlm/. CPU fallback available but slow; CUDA strongly recommended.
