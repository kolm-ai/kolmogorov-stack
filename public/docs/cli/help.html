kolm help

Print general help or help for a specific verb. kolm help <verb> is the same as kolm <verb> --help.

Usage

Run kolm help --help for the authoritative flag reference and current sub-verb list. This page is a discoverability stub — the CLI is the source of truth.

kolm help --help
