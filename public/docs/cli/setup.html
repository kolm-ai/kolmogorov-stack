kolm setup

Set up shell integrations: completion, aliases, environment helpers.

Usage

kolm setup [--shell <bash|zsh|fish|powershell>] [--alias]

Flags

FlagDefaultDescription
--shell autoTarget shell. Auto-detected when omitted.
--aliasoffInstall convenience aliases (k, kc, kr).

Examples

kolm setup
kolm setup --shell zsh --alias

Notes

Idempotent. Re-running upgrades the installed snippet in place. Use kolm completion for one-shot completion script output.
