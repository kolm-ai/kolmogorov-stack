kolm registry

Push, pull, list, or sign artifacts in a registry (public hub or private).

Usage

kolm registry  [args]

Flags

FlagDefaultDescription
push subPush an artifact to the registry.
pull subPull an artifact by slug (also kolm pull).
listsubList artifacts in the active registry.
sign subSign an artifact with the active tenant key.
info subShow metadata for one artifact: hash, provenance, k-score.
--registry public hubRegistry URL. Honors $KOLM_REGISTRY_URL.

Examples

kolm registry push ./inbox-router.kolm --tag inbox/router:1.4.0
kolm registry list --registry registry.acme.internal
kolm registry info acme/inbox-router:1.4.0

Notes

Every push embeds a content-addressed hash plus a signature from the active tenant key. The public hub is opt-in. Private registries can be self-hosted with the kolm-registry image.
