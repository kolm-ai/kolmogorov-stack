Docs / CLI / hardware
CLI contract $ kolm hardware --help $ kolm hardware --json

Probe local GPUs and report VRAM, native dtypes, supported quants, vendor and driver. Same envelope the Account dashboard at /account/hardware renders.

Operational proof

The probe is read-only; no auth, no tenant scoping. Use the supported_methods list to filter kolm compile --target <...> picks.

kolm hardware

What this box can actually run. Live VRAM, native dtypes, supported quant targets, vendor and driver — in plain text or JSON.

Usage


kolm hardware              # human-readable summary of the primary GPU
kolm hardware --json       # full JSON: primary + every detected device
kolm hardware --all        # human-readable summary of every device

Flags

FlagDefaultDescription
--jsonoffemit the full /v1/hardware envelope as JSON
--alloffprint one block per detected device, not just the primary
--quietoffonly print the primary device name and VRAM (single line, scriptable)

Response shape


{
  "ok":      true,
  "primary": {
    "vendor":             "nvidia",
    "name":               "NVIDIA GeForce RTX 5090",
    "vram_gb":            31.8,
    "compute_capability": "12.0",
    "driver_version":     "596.21",
    "native_dtypes":      ["nvfp4", "fp8", "fp16", "bf16", "int8", "int4"],
    "supported_methods":  ["fp16", "bf16", "fp8", "nvfp4",
                           "gguf-q8", "gguf-q6k", "gguf-q5km",
                           "awq-4bit", "gptq-4bit", "gguf-q4km",
                           "gguf-iq4xs", "int4", "exl2", "mlx-4bit", "hqq",
                           "gguf-q3km", "gguf-iq3xxs", "gguf-q2k", "gguf-iq2xs"]
  },
  "all":     [ { /* ... per-GPU detail ... */ } ]
}

Examples


# human-readable
kolm hardware

# extract just the VRAM for scripting
kolm hardware --json | jq -r '.primary.vram_gb'

# list every supported quant target
kolm hardware --json | jq -r '.primary.supported_methods[]'

# pair with fit: pick the highest-quality target that fits
kolm hardware --json | jq -r '.primary | "--vram-gb " + (.vram_gb|tostring)' \
  | xargs kolm fit --params-b 7 --pick-best

Notes

The probe calls into src/forge-hardware.js · detectHardware(). NVIDIA detection uses nvidia-smi if present; otherwise it falls back to a CPU-only envelope. AMD ROCm and Apple Metal detection are best-effort — please file a bug at github.com/kolm-ai/kolm if your device shows up wrong.

The same route powers the Account dashboard auto-fill for VRAM; you should never have to type your VRAM into the quantize wizard.
