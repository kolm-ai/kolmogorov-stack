kolm artifact

Build deployment-ready bundles. The primary sub-verb is `airgap`, which produces a tar.gz of the kolm runtime + your local checkout for offline install.

Usage

kolm artifact airgap --out <path.tar.gz> [--with-node-modules] [--with-wheels] [--with-models] [--force] [--json]

Flags

FlagDefaultDescription
airgaprequiredSub-verb. Builds a tar.gz containing the kolm runtime + checkout suitable for offline install.
--out requiredDestination path. Must end in .tar.gz or .tgz.
--repo-root autoSource repo root. Defaults to the directory where the kolm CLI lives.
--models-dir noneOptional models directory to include when paired with --with-models.
--with-node-modulesoffInclude node_modules/ (large, ~300 MB). Use for fully offline install.
--with-wheelsoffInclude any wheels/ directory (Python wheels for offline pip install).
--with-modelsoffInclude the models directory passed via --models-dir.
--forceoffOverwrite the destination if it already exists.
--jsonoffPrint the result envelope as JSON instead of human text.

Examples

kolm artifact airgap --out /tmp/kolm-airgap.tar.gz
kolm artifact airgap --out kolm.tgz --with-node-modules --with-wheels
kolm artifact airgap --out /tmp/kolm.tar.gz --with-models --models-dir ~/models --json

Notes

Produces a deterministic file list (sorted, sha256 per file) plus MANIFEST.json, BUNDLE-README.md, and VERSION. The bundle README contains the offline install quick-start. Default size is ~30 MB compressed; --with-node-modules pushes that to ~300 MB. The HTTP endpoint /v1/bundle/airgap returns 501 bundle_airgap_requires_cli - air-gap bundles are CLI-only by design.
