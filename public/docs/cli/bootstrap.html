kolm bootstrap

Bootstrap a fresh cluster or container with the kolm runtime + a starter set of artifacts.

Usage

kolm bootstrap [--with-artifacts] [--with-models] [--out <dir>]

Flags

FlagDefaultDescription
--with-artifactsoffInclude a starter set of community artifacts.
--with-modelsoffPre-pull common base models (large download).
--out ~/.kolmBootstrap target directory.

Examples

kolm bootstrap
kolm bootstrap --with-artifacts --with-models

Notes

Idempotent. Safe to re-run on an existing install — files are added but never overwritten.
