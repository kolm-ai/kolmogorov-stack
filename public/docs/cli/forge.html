kolm forge

Umbrella verb for the Forge pipeline: hardware detection, fit checks, model inspection, MoE expert routing, merging, serving, and benchmarking.

Usage

kolm forge <sub-verb> [flags]

Flags

FlagDefaultDescription
hardware-Auto-detect local GPU/CPU and report VRAM/RAM (alias for kolm hardware).
fit-Check whether a target model will fit on detected hardware before compile (alias for kolm fit).
inspect-Inspect a .kolm or .safetensors file (alias for kolm inspect).
experts-List or route Mixture-of-Experts components (alias for kolm experts).
merge-Merge two artifacts (alias for kolm merge).
serve-Start a local inference server (alias for kolm serve).
bench-Benchmark an artifact (alias for kolm bench).

Examples

kolm forge hardware
kolm forge fit --model meta-llama/Llama-3.1-8B --target cuda
kolm forge inspect ./out/recipe.kolm
kolm forge serve ./out/recipe.kolm --port 8787

Notes

The forge umbrella is for discoverability — every sub-verb is also reachable directly. Add --json to any sub-verb for machine-readable output.
