kolm make

Unified pipeline verb: takes a recipe and runs distill, quantize, compile, eval, and (optionally) deploy. Like make for AI models.

Usage

kolm make <recipe.toml> [--target <slug>] [--deploy]

Flags

FlagDefaultDescription
requiredPath to the recipe file.
--target cudaCompile target. See kolm compile --help for the full list.
--deployoffPush to the deploy hook after a green compile.
--gate 0.85K-Score floor for the gate.

Examples

kolm make ./recipes/support-bot.toml
kolm make ./recipes/support-bot.toml --target metal --deploy

Notes

Wrapper over kolm distill + kolm quantize + kolm compile + kolm eval. Use the explicit verbs for per-step overrides.
