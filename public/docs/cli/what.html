CLI contract $ kolm what --help $ kolm what --json

Every reference page is generated from the command source, keeps usage and examples close to the product, and links into the same API, account, and artifact workflow.

Operational proof

Use this verb inside the capture, dataset, distill, compile, verify, or runtime loop. Scripts can pin JSON output, humans get the same flow in the account console and TUI.

kolm what

One-screen snapshot. Artifacts, captures, jobs, recommended next steps.

Usage


kolm what [--json]

Flags

FlagDefaultDescription
--jsonoffstructured envelope

What it reads

  • ~/.kolm/artifacts/ for local artifact count
  • the local capture store for namespace counts
  • ~/.kolm/jobs/ for running and recent jobs
  • your config for tenant / base

Examples


kolm what
# artifacts: 5 captures: 2348 (4 namespaces, 1 distill-ready)
# jobs: 1 running, 3 done
# next: kolm distill --namespace claims-router

kolm what --json

Notes

The next-steps section ranks 1-3 high-value actions and prints the exact command to copy-paste. See kolm next for the same logic surfaced standalone.
