kolm shell-init

Print shell-init snippet (PATH, completion, env) for sourcing in ~/.bashrc / ~/.zshrc / Fish config.

Usage

kolm shell-init [--shell <bash|zsh|fish|powershell>]

Flags

FlagDefaultDescription
--shell autoTarget shell. Auto-detect when omitted.

Examples

kolm shell-init
kolm shell-init --shell zsh >> ~/.zshrc

Notes

For an interactive installer use kolm setup. For completion-only output use kolm completion.
