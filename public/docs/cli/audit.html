kolm audit

View, filter, and stream the audit log. Every state-changing CLI verb and HTTP request is recorded.

Usage

kolm audit [--since <ts>] [--actor <id>] [--action <slug>] [--json] [--tail]

Flags

FlagDefaultDescription
--since 24hShow events since this timestamp or relative duration.
--actor allFilter to a specific tenant id or API key prefix.
--action allFilter to one action class (e.g. compile, auth.signup).
--jsonoffJSON line stream.
--tailoffFollow the log (like tail -f).

Examples

kolm audit --since 1h
kolm audit --action compile --json | jq .
kolm audit --tail

Notes

Mirrors /v1/account/audit-log. Retention is plan-tiered: free 30d, pro 90d, business 365d, enterprise 2555d (7y).
