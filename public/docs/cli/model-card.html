kolm model-card

Render a HuggingFace-style MODEL_CARD.md for an artifact.

Usage

kolm model-card <artifact.kolm> [--out <path>]

Flags

FlagDefaultDescription
requiredArtifact to describe.
--out stdoutDestination file.

Examples

kolm model-card ./out/recipe.kolm --out MODEL_CARD.md

Notes

Covers intended use, training data, evals, base model attribution, license, and limitations. For compliance-oriented review packs use kolm passport export --format compliance instead.
