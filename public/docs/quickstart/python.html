Python SDK quickstart.

The kolm Python SDK is a sync-and-async client for Python 3.10+. It drops into FastAPI route handlers, LangChain pipelines, Jupyter notebooks, and any synchronous script. This page takes you from pip install to your first captured request in five minutes.

Install

Install from source while the PyPI package is still pending publish. The install-from-source path uses pip's VCS support.

pip install git+https://github.com/kolm-ai/kolm@main#subdirectory=sdk/python

# or, once PyPI publish lands:
pip install git+https://github.com/kolm-ai/kolm@main#subdirectory=sdk/python

Requires Python 3.10 or newer. The package ships sync and async clients side by side.

Authenticate

The SDK reads KOLM_API_KEY from the environment by default. Get a key from the CLI or from the dashboard.

# option 1: log in with the CLI
kolm login

# option 2: set the env var directly
export KOLM_API_KEY=ks_your_key_here
export KOLM_BASE_URL=https://kolm.ai   # optional, defaults to https://kolm.ai

Sign up for a free key at kolm.ai/signup if you do not have one yet.

First capture

Create first_capture.py and run it. The snippet captures a single request into the quickstart namespace, then prints the captured row's id.

from kolm import Kolm

kolm = Kolm()   # reads KOLM_API_KEY + KOLM_BASE_URL

me = kolm.whoami()
print("authed as", me["tenant_id"])

captured = kolm.capture.log(
    namespace="quickstart",
    input={"prompt": "What is the capital of France?"},
    output={"reply":  "Paris"},
    meta={"source": "python-quickstart"},
)
print("captured id:", captured["id"])

Run it:

python first_capture.py
# authed as tenant_f33c240034c5
# captured id: cap_a1b2c3d4...

The async client has the same shape:

import asyncio
from kolm import AsyncKolm

async def main():
    async with AsyncKolm() as kolm:
        captured = await kolm.capture.log(
            namespace="quickstart",
            input={"prompt": "Hello"},
            output={"reply":  "World"},
        )
        print(captured["id"])

asyncio.run(main())

First deploy (outline)

Once you have a few thousand captures in the namespace, the deploy path looks like this from a Python script.

job = kolm.compile.create(
    namespace="quickstart",
    target="qwen2.5-3b-instruct-int4",
    epochs=3,
)
print("compile job:", job["id"])

artifact = kolm.compile.wait(job["id"])
print("artifact cid:", artifact["cid"], "k_score:", artifact["k_score"])

deploy = kolm.deploy.create(
    artifact_cid=artifact["cid"],
    runtime="vllm",
    endpoint="https://gpu.internal:8000",
)
print("deployed:", deploy["endpoint"])

The blocking compile.wait call polls until the job finishes or fails. Use the async variant inside an event loop.
