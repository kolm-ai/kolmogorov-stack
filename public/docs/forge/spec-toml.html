Forge

spec.toml.

A kolm artifact is reproducible to the byte if and only if it has a single, versioned source of truth. That source is spec.toml: base model, capture pointers, teacher council, recipe, LoRA targets, k-score floor, governance policy. The file lives in your repo; kolm compile reads it and emits a signed .kolm.

Minimal spec

# spec.toml
name        = "support-tier-1"
version     = "0.3.0"
description = "Customer support, tier 1, English, business hours."

[base]
model = "Qwen/Qwen2.5-7B-Instruct"

[captures]
namespace = "prod-support-en"
since     = "2026-04-01"
filter    = { language = "en", policy = "allow" }

[distill]
recipe   = "trinity"
budget_usd = 200

[gate]
k_score_floor = 0.84

That spec compiles. Everything else has a default.

Sections

SectionPurpose
[base]Pin the base model + revision. SHA pinning is recommended for compliance.
[captures]Which captures feed the run. Namespace, time window, filter predicates.
[distill]Recipe, budget cap, teacher overrides, progressive stages.
[lora]Rank, alpha, dropout, target modules. Defaults come from the recipe.
[gate]k-score floor and per-rubric floors that must pass for the artifact to be signed.
[govern]Lifecycle approvals, drift thresholds, compliance tags. See lifecycle.
[export]Optional auto-export targets after compile (GGUF, AWQ, EXL2, MLX).
[publish]Optional auto-publish targets after sign (Hugging Face, model registry).

Custom council example

[distill]
recipe      = "custom"
budget_usd  = 350
progressive = ["warm", "sharpen", "polish"]

[distill.teachers]
a = { model = "anthropic/claude-opus-4-7",  weight = 1.0 }
b = { model = "openai/gpt-4o",              weight = 1.0 }
c = { model = "deepseek/deepseek-v3",       weight = 0.8 }

[distill.judge]
model = "google/gemini-2.5-pro"   # must be a different lineage
floor = 0.65

[gate]
k_score_floor   = 0.84
rubric_floors   = { factuality = 0.90, policy = 1.00 }
shadow_window_h = 24

Validate before you compile

kolm spec validate spec.toml
# OK: 7 sections, 0 errors, 2 warnings
# warn: gate.k_score_floor=0.84 is below recipe trinity default 0.86
# warn: distill.budget_usd=350 exceeds tenant soft cap $300 (allowed; will prompt)

kolm spec hash spec.toml
# sha256: 6f3a...c281

The hash is the same hash that lands in the artifact manifest at compile time — check it into source control.
