Compile

GGUF deep-dive.

GGUF is the format llama.cpp, ollama, lm-studio, and a growing list of small-host runtimes consume. It bundles weights, tokenizer, and metadata into one mmap-friendly file. kolm's GGUF builder runs the importance-matrix calibration over your own captures, picks the right K-quant / I-quant per layer, and splits the file when it crosses 50 GB so distribution does not break.

K-quants vs I-quants

FamilyExamplesWhat it is
K-quantsq4_k_m, q5_k_m, q6_kBlock-wise quantization with per-block scales and a super-block; great quality/size on most hardware.
I-quantsiq4_xs, iq3_xxs, iq2_mCodebook-based; smaller files and better low-bit quality, slower on CPUs without AVX2.
Legacyq4_0, q8_0Older block formats; q8_0 still useful as a quality reference.

Importance matrix

I-quants and the per-tensor K-quant variants benefit from an importance matrix that tells the quantizer which weights matter most for your specific traffic. kolm computes the matrix from a sample of the same captures the artifact was distilled from:

kolm export art_a1b2c3d4 --format gguf-q4_k_m --imatrix auto

# Equivalent to:
# 1. sample 256 captures from the artifact's namespace
# 2. run a forward pass per capture to record activation magnitudes
# 3. write the imatrix to weights.imatrix
# 4. quantize each layer using the per-layer importance

For air-gapped builds, supply an imatrix file pre-built from a representative sample: --imatrix ./calib.imatrix.

Split files

Hugging Face refuses single files over 50 GB; some clients refuse over 4 GB. kolm splits big GGUFs automatically:

kolm export art_dsr1_32b --format gguf-q4_k_m --split-max-size 4G

# Outputs:
# art_dsr1_32b-q4_k_m-00001-of-00005.gguf
# art_dsr1_32b-q4_k_m-00002-of-00005.gguf
# ...
# art_dsr1_32b-q4_k_m-00005-of-00005.gguf

# llama.cpp loads the first split; the rest are mmapped transparently:
llama-cli -m art_dsr1_32b-q4_k_m-00001-of-00005.gguf -p "hello"

Picking a variant

  • Default. q4_k_m is the universally safe pick.
  • Quality first. q5_k_m or q8_0 when the host can spare the bytes.
  • Smallest footprint. iq4_xs or iq3_xxs; verify k-score after.
  • CPU only. Stick with K-quants on older CPUs without AVX2.
