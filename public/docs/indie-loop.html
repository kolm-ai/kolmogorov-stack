The 8 commands

Run them in order. Each prints what the next one needs.

Substitute your own namespace name and target device. Everything else is verbatim.

1

Install + sign in

out: ~/.kolm/credentials

npm i -g @kolm/cli && kolm login

2

Create a namespace

out: namespace id

kolm namespace create support-bot --primary-provider anthropic

3

Capture real traffic

in: prod gateway calls · out: capture lake rows

Point your app at api.kolm.ai/v1/chat/completions with header X-Kolm-Namespace: support-bot. Run ~200 calls.

4

Approve the captures

in: capture browser · out: approved rows

kolm captures approve --namespace support-bot --status pending --all

5

Compile

in: approved captures · out: out.kolm + K-Score

kolm compile --namespace support-bot --target gguf-q4km --kscore-gate 0.90

6

Test on the device

in: out.kolm + device · out: pass/fail + latency

kolm devices add edge-01 --type ssh --host 10.0.0.7 && kolm test-device out.kolm --device edge-01

7

Deploy

in: out.kolm + device · out: live endpoint

kolm deploy out.kolm --device edge-01 --port 8766

8

Verify the receipt

in: deploy id · out: signed receipt

kolm verify $(kolm fleet status --tag edge-01 --json | jq -r .receipt)
