LlamaIndex Python

LlamaIndex Python uses the official openai and anthropic Python SDKs internally. Pass api_base= on the LLM constructor and every higher-level abstraction (query engine, chat engine, agent) follows. Set Settings.llm once and the rest of the pipeline routes through the local kolm proxy automatically.

Install

pip install llama-index llama-index-llms-openai llama-index-llms-anthropic
npm i -g github:sneaky-hippo/kolm-stack

Start the proxy

kolm capture --provider openai --as docs-rag --namespace docs-rag
# proxy listening on http://127.0.0.1:8787

OpenAI LLM

from llama_index.llms.openai import OpenAI
from llama_index.core import Settings
import os

Settings.llm = OpenAI(
 model="gpt-4o-mini",
 api_key=os.environ["OPENAI_API_KEY"],
 api_base="http://127.0.0.1:8787/v1",
 default_headers={
 "x-upstream-api-key": os.environ["OPENAI_API_KEY"],
 "x-kolm-namespace": "docs-rag",
 },
)

response = Settings.llm.complete("summarize: ...")
print(response.text)

Anthropic LLM

from llama_index.llms.anthropic import Anthropic
from llama_index.core import Settings
import os

Settings.llm = Anthropic(
 model="claude-3-5-sonnet-latest",
 api_key=os.environ["ANTHROPIC_API_KEY"],
 base_url="http://127.0.0.1:8787",
 default_headers={
 "x-upstream-api-key": os.environ["ANTHROPIC_API_KEY"],
 "x-kolm-namespace": "docs-rag",
 },
)

RAG and agents

from llama_index.core import VectorStoreIndex, SimpleDirectoryReader

documents = SimpleDirectoryReader("./docs").load_data()
index = VectorStoreIndex.from_documents(documents)

query_engine = index.as_query_engine()
response = query_engine.query("what is a kolm artifact?")
print(response)

Every LLM call inside the query engine now routes through the proxy. Embeddings are configured separately via Settings.embed_model; if you want those captured too, point the embedding model at the same base URL.

Env-var shortcut

import os

os.environ["OPENAI_BASE_URL"] = "http://127.0.0.1:8787/v1"
os.environ["ANTHROPIC_BASE_URL"] = "http://127.0.0.1:8787"

# now: from llama_index.llms.openai import OpenAI; OpenAI() picks the env

Verify the proxy sees traffic

kolm capture status --namespace docs-rag
# pairs captured: 412 / 1000
# pairs until distill: 588
