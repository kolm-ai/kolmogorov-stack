LangChain Python

LangChain Python wraps the official openai and anthropic Python SDKs. Pass base_url= on construction (or set the env var) and the chain talks to the local kolm proxy instead of the upstream provider. Streaming, tool calls, and structured output pass through unchanged.

Install

pip install langchain langchain-openai langchain-anthropic
npm i -g github:sneaky-hippo/kolm-stack

Start the proxy

kolm capture --provider openai --as triage-chain --namespace triage
# proxy listening on http://127.0.0.1:8787

ChatOpenAI

from langchain_openai import ChatOpenAI
import os

model = ChatOpenAI(
 model="gpt-4o-mini",
 api_key=os.environ["OPENAI_API_KEY"],
 base_url="http://127.0.0.1:8787/v1",
 default_headers={
 "x-upstream-api-key": os.environ["OPENAI_API_KEY"],
 "x-kolm-namespace": "triage",
 },
)

result = model.invoke("classify: order delayed")
print(result.content)

ChatAnthropic

from langchain_anthropic import ChatAnthropic
import os

model = ChatAnthropic(
 model="claude-3-5-sonnet-latest",

 base_url="http://127.0.0.1:8787",
 default_headers={
 "x-upstream-api-key": os.environ["ANTHROPIC_API_KEY"],
 "x-kolm-namespace": "triage",
 },
)

Tools and structured output

LangChain's bind_tools, with_structured_output, and the async streaming APIs all pass through unchanged. The proxy forwards the upstream response body byte-for-byte; only the metadata sidecar (latency, namespace, token counts) is appended to the local lake.

Env-var shortcut

If you do not want to touch the construction args, set the env vars before importing:

import os

os.environ["OPENAI_BASE_URL"] = "http://127.0.0.1:8787/v1"
os.environ["ANTHROPIC_BASE_URL"] = "http://127.0.0.1:8787"

# now: from langchain_openai import ChatOpenAI; ChatOpenAI() picks the env

The official openai and anthropic Python SDKs honor these env vars at construction time. Set them before from langchain_openai import ....

Verify the proxy sees traffic

kolm capture status --namespace triage
# pairs captured: 38 / 1000
# pairs until distill: 962

kolm distill --namespace triage # once the threshold trips

Async

import asyncio
from langchain_openai import ChatOpenAI

async def main():
 model = ChatOpenAI(model="gpt-4o-mini", base_url="http://127.0.0.1:8787/v1")
 async for chunk in model.astream("hello"):
 print(chunk.content, end="")

asyncio.run(main())
