Make.com integration

Make.com (formerly Integromat) has no native kolm module yet. You call a deployed artifact via the generic HTTP module. Bearer auth in the header, JSON body in, JSON envelope out. The same shape as the REST API.

Prerequisites

  • A kolm API key. Get one from /signup or via kolm signup --email you@company.com.
  • An artifact you have already compiled and published. The walkthrough below uses <owner>/<name> as the handle.
  • A Make.com account at the free tier or above.

Five minutes

Step 1

Add an HTTP > Make a request module

In a new or existing scenario, click the plus, search for HTTP, and pick Make a request. This is the generic HTTP client; you will set headers, body, and parse the JSON response yourself.

Step 2

Configure the request

URL: https://kolm.ai/v1/artifacts/<owner>/<name>/run
Method: POST
Headers: Authorization: Bearer kolm_...
 Content-Type: application/json
Body type: Raw
Content type: JSON (application/json)
Request content:
{
 "input": "{{1.text}}"
}

Replace <owner>/<name> with your artifact handle. The {{1.text}} is the standard Make mapping that pulls a value from the previous module.

Step 3

Parse the response

Toggle Parse response to Yes so Make treats the body as JSON. Downstream modules can then map {{2.output}}, {{2.k_score}}, and other fields straight out.

Step 4

Test the scenario

Click Run once. The HTTP module turns green and the bubble shows the parsed envelope. If you see 401, the bearer header is wrong. If 404, the handle has a typo. If 422, the artifact is missing a required input field.

Step 5

Save and schedule

Save the scenario. Set the trigger (cron, webhook, watch a folder) and turn the scheduler on. Every run logs to the Make execution history; the request and the parsed body are both captured on the Make side.

Webhook trigger

If you want the kolm artifact to fire from an upstream event, add a Webhooks > Custom webhook as the first module. Make assigns a unique URL; point your upstream system at that URL, and every POST kicks off the scenario.
