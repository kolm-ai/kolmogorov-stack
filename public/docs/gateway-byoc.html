kolm.ai / docs / gateway-byoc

Wrapper / Gateway

Deploy where your data lives. No bytes to kolm.

BYOC is for deployments that cannot send data out - regulated industries, classified networks, sovereign-cloud requirements. kolm artifact airgap produces a self-contained tarball; kolm passport export --format compliance produces a compliance manifest that lists every dependency + every receipt-signing decision.

Produce the bundle

kolm artifact airgap --output kolm-gateway-airgap-2026-05-26.tar.gz \
  --include image,helm,migrations,docs,checklist

# Inspect:
tar -tzf kolm-gateway-airgap-2026-05-26.tar.gz | head
# kolm-gateway-airgap/
#   image/kolm-gateway-latest.tar
#   helm/kolm-gateway-chart-0.1.0.tgz
#   migrations/001_initial.sql
#   docs/compliance.pdf
#   CHECKLIST.md

Compliance passport

kolm passport export --format compliance > compliance.json

# Lists:
#   - every dep + license + version
#   - every receipt-signing decision (key rotation history)
#   - every namespace + its routing chain
#   - every PII mode in use
#   - every capture lake destination
#   - SBOM (CycloneDX)

Hand to your model-risk-review team. Bundled with the airgap tarball by default.

Procurement questionnaires

For SIG / CAIQ / SOC 2: kolm procurement export --format sig|caiq|all generates pre-filled answers from the live deployment config. See procurement.
