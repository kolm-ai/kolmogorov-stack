Reference

Ship gate.

The 52-check launch gate that wraps scripts/release-verify.cjs. ALL checks must pass to ship a kolm.ai release. Every check has a stable id, the surface it belongs to, what passing means, and a one-line fix when it fails. Run with npm run ship:gate or kolm ship --check.

One-line example

npm run ship:gate
# → 52 checks across 7 surfaces
# → gateway 8/8 / capture 7/7 / compile 6/6 / run 8/8
# → account 6/6 / infra 7/7 / perf 5/5 / docs 5/5
# → OK (52/52, 411s)

How to run it

# Full gate (CI before a release)
npm run ship:gate

# JSON output for dashboards
npm run ship:gate -- --json > ship-report.json

# Skip a surface during incident fix-forward (use sparingly; recorded in audit)
npm run ship:gate -- --skip=perf --reason "perf-fix-rolling"

# Per-surface re-run after a fix
npm run ship:gate -- --only=run

Exit codes

  • 0 — all gated checks pass.
  • 1 — one or more required checks failed; do not ship.
  • 2 — gate itself errored (test runner crashed, no network, etc).
  • 3 — user-supplied --skip applied without --reason (skips are recorded but require justification).

JSON shape

{
  "version": 1,
  "ok": false,
  "exit_code": 1,
  "summary": { "total": 52, "ok": 51, "fail": 1, "skipped": 0, "duration_s": 411 },
  "surfaces": {
    "gateway": { "ok": 8, "fail": 0 },
    "capture": { "ok": 7, "fail": 0 },
    "compile": { "ok": 6, "fail": 0 },
    "run":     { "ok": 7, "fail": 1 },
    "account": { "ok": 6, "fail": 0 },
    "infra":   { "ok": 7, "fail": 0 },
    "perf":    { "ok": 5, "fail": 0 },
    "docs":    { "ok": 5, "fail": 0 }
  },
  "fails": [
    { "id": "run.canary", "detail": "monitor window failed: k-score 0.62 < floor 0.70" }
  ]
}
Constraints. Skipping a surface (--skip=<surface>) requires --reason and is recorded in the audit chain. The CI workflow rejects PRs that skip without a justification matching incident-YYYY-MM-DD-*. Performance numbers come from scripts/bench-prod.cjs against a fresh tenant; rerun with --n=20 to widen the sample if a single outlier blocks the gate.
