Rust SDK -- see /docs/quickstart/rust.

The canonical Rust SDK quickstart lives at /docs/quickstart/rust. This page exists so any link to /docs/sdk/rust resolves to a real page instead of a 404.

The Rust SDK ships from sdk/rust/ in the public repo as crate kolm v0.1.0. Add to Cargo.toml: kolm = { git = "https://github.com/sneaky-hippo/kolmogorov-stack", branch = "main" } until the crates.io publication lands.
