Node SDK -- see /docs/quickstart/node.

The canonical Node SDK quickstart lives at /docs/quickstart/node. This page exists so any link to /docs/sdk/node resolves to a real page instead of a 404.

The Node SDK ships from sdk/node/ in the public repo. npm install @kolm/sdk installs the published package once it is on the npm registry; in the meantime, install from source with npm install github:kolm-ai/kolmogorov-stack#main:sdk/node.
