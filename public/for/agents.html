The problem

Why agent loops don’t scale to production.

Cost

An 18-hop loop costs 18× a single-pass call.

Tool calls, intermediate reasoning, self-correction — every step is a billable round trip. For a query you’ve seen before, you’re paying the exploration cost on every replay.

Latency

Serial tool calls add seconds the user feels.

A 4-tool sequential agent at 800ms per hop is a 4-second wait. A distilled specialist that learned the final synthesis returns in 60ms.

Determinism

The same input takes a different path every run.

Eval suites are flaky because the agent re-explores. Kolm’s compiled trace replays deterministically against the cached path — the eval row either matches a recorded trace or it falls back to the live loop.

Auditability

A 30-step trace is not a reviewable artifact.

Your reviewers need a frozen behavior they can pin and re-run. Kolm gives you a single signed model whose input/output the compliance team can sample-test forever.

What kolm changes

From trace capture to compiled specialist.

Kolm’s trace-compile path takes a recorded agent run and produces a workflow IR that returns the original final output on cache hit. The specialist is a real model — not a lookup table — trained to skip the search-and-reason loop for inputs near the trace distribution.

Step 01

Capture the trace

Your agent calls run through the kolm gateway; each LLM hop, each tool call, and the final output are logged into a trace with a stable ID.

$ kolm trace capture --namespace agent.prod \
  --max-hops 30 --tool-calls included
Step 02

Compile trace to replay-IR

Kolm’s trace-compile builds an IR whose cache key is (user-input, tool-set) and whose payload is the validated final output. Verification mode confirms the IR reproduces the recorded path within tolerance.

$ kolm trace compile trace_4b7c...
$ kolm trace verify trace_4b7c... --coverage 0.97
ok · matches 97/100 · mismatches 3
Step 03

Distill the synthesis

For inputs that fall outside the cache (the long tail), a distilled student trained on the agent’s synthesis step answers in one pass. Tools still run when they must; the LLM hop count drops 80–95%.

$ kolm distill --from agent.prod \
  --student qwen2.5-7b --quantize int4 --sign
ok · agent-specialist-v2.kolm
