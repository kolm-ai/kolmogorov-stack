kolm.ai / integrations / langchain

LangChain ChatOpenAI, with kolm underneath.

LangChain's ChatOpenAI already accepts a custom base_url. Point it at kolm.ai and every node in every chain – LangGraph, LangServe, RetrievalQA – runs the same way. Captures, the redaction membrane, and signed receipts come along for free.

How it works in one paragraph

LangChain's ChatOpenAI (Python langchain-openai, JS @langchain/openai) takes base_url / configuration.baseURL at construction. kolm.ai speaks the OpenAI Chat Completions wire format, so swapping the URL is the entire integration. Tool calling, streaming, structured outputs, callbacks, and LangSmith tracing keep working. Every call lands as a capture row tagged with the LangChain run-id so you can correlate kolm artifacts back to LangSmith traces.

Python – one line

Construct ChatOpenAI with base_url + api_key. Drop it into any chain.

python – langchain-openai >= 0.2
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate

llm = ChatOpenAI(
    base_url="https://kolm.ai/v1",
    api_key="ks_live_...",
    model="qwen-3.5-27b",
)

prompt = ChatPromptTemplate.from_messages([
    ("system", "You summarize pull requests in one sentence."),
    ("user", "{diff}"),
])

chain = prompt | llm
print(chain.invoke({"diff": "..."}).content)

JavaScript / TypeScript – one line

Same surface for @langchain/openai. Pass configuration.baseURL.

node – @langchain/openai
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";

const llm = new ChatOpenAI({
  model: "qwen-3.5-27b",
  apiKey: process.env.KOLM_API_KEY,
  configuration: { baseURL: "https://kolm.ai/v1" },
});

const prompt = ChatPromptTemplate.fromMessages([
  ["system", "You summarize pull requests in one sentence."],
  ["user", "{diff}"],
]);

const chain = prompt.pipe(llm);
console.log((await chain.invoke({ diff: "..." })).content);
