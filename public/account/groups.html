Compile groups.

Bundle namespaces that should compile as one model. kolm compile --group support-all folds every member namespace into a single training run, then threads the resolved namespace list + per-namespace pair counts into the passport so verifiers see exactly which corpora went in.

CLI mirror

Every action on this page has a CLI equivalent. Useful for scripting + CI.

# list groups kolm group list # create a group kolm group create support-all --namespaces "retail-support,b2b-support,billing-support" # inspect one kolm group show support-all # add or drop a namespace kolm group update support-all --add-namespace partners-support kolm group update support-all --remove-namespace billing-support # compile every namespace in the group as one model kolm compile --group support-all --target gguf-q4km # soft-delete kolm group delete support-all

The passport for a group-driven compile records source: "group", the group slug + name, the resolved namespace list, and per-namespace pair counts — so a downstream verifier can confirm the model was trained on the exact corpora claimed. See docs/gateway-namespaces.
