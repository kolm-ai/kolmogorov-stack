Pipelines

W821 pipeline orchestrator. Compose classifier + specialist artifacts into a single named pipeline. Run via POST /v1/pipelines/:id/run or CLI kolm pipeline run pl_… --input-json '{…}'.

Read the docs
Create a new pipeline

The web UI for creating a pipeline ships in a follow-up wave. For now, use either:

  • CLI: kolm pipeline create --name <name> --file kolm.pipeline.yaml
  • HTTP: POST /v1/pipelines with body {name, yaml}

YAML schema (W821 friendly format, distinct from W738 cid-pinned format):

version: v1
classifier: ./classifier.kolm
routes:
  - match: {intent: refund}
    artifact: ./specialists/refund.kolm
  - match: {intent: status}
    artifact: ./specialists/status.kolm
  - default: true
    artifact: ./specialists/fallback.kolm
