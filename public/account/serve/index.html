What a pod reports

Pods register with the control plane on boot and emit a heartbeat every 30 seconds. If the route is unavailable, the pane shows an empty state and the CLI is still the source of truth.

SignalSource
endpointPOST /v1/chat/completions (OpenAI-compatible)
healthGET /health on each pod
latency / throughputGET /metrics Prometheus exposition
runtimeauto-detected by serve-autodetect from artifact format + hardware
artifact idpassed through from the .kolm manifest on pod boot

CLI parity

Every action on this page is mirrored by the CLI. The CLI is the source of truth for air-gapped deployments where this dashboard is not reachable.

kolm serve <artifact> (auto-detect runtime + hardware)
kolm serve <artifact> --runtime vllm --port 8000
kolm serve list --json
kolm serve stop <pod-id>
kolm serve logs <pod-id> --tail 100
