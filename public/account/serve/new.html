Forge

Serve a .kolm.

Emit a Docker run command or a Kubernetes Deployment+Service manifest for a compiled artifact and a runtime target. Same artifact, six runtime backends; the manifest pins the GPU resource request and the artifact mount point. Backed by POST /v1/serve.

CLI parity

Same envelope from the terminal: kolm serve <artifact> --runtime vllm --docker or --k8s. The CLI prints the same manifest; this page just calls /v1/serve and renders it with a copy button.
