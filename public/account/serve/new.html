CLI parity

Every output on this page is equivalent to a single CLI command. Use the CLI in scripts and CI; use this page when you want a one-off copy.

kolm serve <artifact> --docker
kolm serve <artifact> --k8s
kolm serve <artifact> --runtime vllm --port 8000 --context-length 4096
