Connectors.

Configured AI providers. Per-provider status pill, capture endpoint, last seen. Add another by following the kolm capture proxy doc.

Provider docs
ProviderStatusCapture endpointLast event
No connectors wired up yet. Pick a provider you already pay for and point its base URL at the kolm capture proxy:

export OPENAI_BASE_URL=https://kolm.ai/v1/capture/openai
export OPENAI_API_KEY=$KOLM_API_KEY
x-upstream-api-key: $PROVIDER_API_KEY

Every call your code makes is then proxied upstream and durably captured. Same calls work for /v1/capture/anthropic, /v1/capture/openrouter, and /v1/capture/gemini.
