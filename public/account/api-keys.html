API keys.

CLI mirror

Everything on this page is available as a kolm CLI command. Pipe-friendly, scriptable, no browser required.

# list every key on the tenant (table)
kolm keys list

# rotate the current key (old key invalidates immediately)
kolm keys rotate

# revoke a specific key by prefix
kolm keys revoke ks_a3f1

# JSON output for scripting
kolm keys list --format json

What rotation does

  • New secret returned exactly once. Stored as a salted SHA-256 hash on the server; the cleartext is not retrievable.
  • Old key stops working immediately. In-flight requests using the old key get a 401 within the next round-trip.
  • Webhooks keep their signing secrets – webhook signing is separate from API key auth (see /docs/webhooks).
  • Audit entry written. A row lands on /account/audit-log with action=key.rotated, your IP, and the new key prefix.
