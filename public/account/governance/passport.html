Runtime passport.

Per-target latency, memory, throughput, and quality delta for every signed artifact. tested means the number came from a real measurement on real hardware; estimated is a memory-fit projection; unsupported is gated by the artifact format.

Docs · Run overview Docs · Hardware

Schema

Each row in the runtime matrix carries the same nine fields. Missing values render as -- rather than being elided.

FieldTypeMeaning
target_idstringruntime + precision + hardware fingerprint
statusenumtested / estimated / unsupported
runtimeenumvllm / llama.cpp / ollama / tgi / sglang / trt-llm / mlx
runtime_versionstringresolved version string at measurement time
precisionenumfp16 / bf16 / int8 / int4 / fp8 / nvfp4 / awq / gptq / gguf_qN_K_M
memory_mbnumberpeak resident memory while serving
latency_p50_ms / p95_msnumberper-token latency at observed concurrency
tok_snumberthroughput in output tokens per second
quality_deltanumberK-Score delta vs the FP16 baseline; negative is a regression
