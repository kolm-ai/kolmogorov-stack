Evidence DAG.

Every signed artifact carries a directed acyclic graph of the evidence that backs it: capture rows, eval reports, teacher fingerprints, student adapters, runtime measurements, signatures, policies, and rights records. Revocation of any node propagates to its descendants.

Edge relationships

Edges are typed; the DAG validator rejects cycles. A revoke on any node marks every descendant needs_review without touching the underlying records.

RelationshipMeaning
derived_fromdownstream node was produced using the upstream node
validated_byupstream node passes a check that the downstream node defines
invalidatesupstream node contradicts the downstream node
supersedesupstream node is a newer canonical version of the downstream node
