Settings.

Per-tenant account settings. Same source of truth as kolm settings (CLI) and the TUI settings view (F). Updates are audited; defaults are returned alongside current values so you can always see what the platform ships out of the box.

Capture & sync

FieldValue
default_namespace
Namespace assigned to new captures when the client doesn't set one.
capture_default_durable
If on, captures require x-kolm-capture-durable to consider the row landed.
sync_push_enabled
Per-tenant kill-switch for outbound sync push from kolm runners.

Privacy

FieldValue
redaction_strictness
strict: 19-class PHI+PII+PCI+secrets. balanced: PHI+secrets only. minimal: secrets only.

Notifications

FieldValue
notifications_email
Email for threshold alerts + drift alerts.
notifications_webpush
WebPush for live threshold alerts in the browser.
key_rotation_warning_days
Show a banner this many days before signing-key expiry.

Locale

FieldValue
locale
IETF tag (e.g. en-US, ja-JP). Currently affects date format only.
timezone
IANA tz database name (e.g. UTC, America/New_York).

Trust packet (workspace)

Procurement-ready bundle of framework controls (SOC 2, HIPAA, EU AI Act, ISO 27001) anchored to your workspace's procurement vault. Per-artifact export lives on the Artifacts page.
