Governance.

Every signed .kolm ships with the evidence to prove what it is, what it was trained on, what it runs on, and what it costs. This hub is the index.

Six surfaces, one artifact

Each tile reads from a route on the same artifact id. Empty states are real: if you have not deployed an artifact yet, the surface will say so and link to the relevant CLI.

R-1 Runtime passport Per-target latency, memory, throughput, and quality delta vs the FP16 baseline. Tested, estimated, or unsupported.
GET /v1/passport/:id
 R-5 Evidence DAG Capture, eval, teacher, student, runtime, signature, policy, rights nodes plus the edges that bind them.
GET /v1/evidence/:id
 R-6 Assurance case Claims and framework controls with evidence pointers, exportable as JSON or PDF for compliance review.
GET /v1/assurance/artifact/:id
 R-7 Drift signals Fallback rate, distribution distance, and volume ratio over a 2σ warn / 3σ alert ladder per namespace.
GET /v1/drift/status
 R-8 Cost displacement Per-namespace frontier baseline vs actual, monthly + cumulative savings, payback period.
GET /v1/savings/displacement
 R-2 Artifact lifecycle State browser across created, signed, deployed, monitored, superseded, revoked, archived.
GET /v1/artifacts/:id/lifecycle

How this is wired

All six surfaces read the same underlying record. There is no separate governance database; the artifact is the governance record.

  • The runtime passport and evidence DAG are persisted inside the .kolm manifest at compile time.
  • Lifecycle transitions land in data/artifacts/<id>/lifecycle.json and are recorded with actor + reason + optional evidence id.
  • Drift signals are computed from the capture lake plus receipt route decisions for the namespace.
  • Cost displacement repeats the same receipt scan against the frozen rate card for the period.
  • The assurance case stitches all of the above into one signed envelope you can hand to an auditor.

CLI parity

Every surface in this hub has a CLI verb that returns the same JSON envelope.

kolm inspect <artifact> --runtime-passport --json
kolm evidence trace <artifact>
kolm assurance export --artifact <id> --format pdf
kolm drift status --namespace support
kolm savings --namespace support --period 30
kolm lifecycle <artifact>
