Forge

Bench all targets.

For one base model and your local VRAM, compute the predicted size + fit for every quantization target this host supports. The "Recommended" row is the highest-quality target that fits with comfortable headroom — the same answer kolm fit --pick-best returns.

CLI parity

Same matrix from the terminal: kolm bench <artifact> --all-targets. kolm fit --params-b 7 --vram-gb 24 --pick-best returns the single "recommended" row.
