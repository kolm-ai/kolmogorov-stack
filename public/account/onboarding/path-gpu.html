Path A · Have a GPU

Install, route, watch a capture land.

Three steps. The CLI installs in seconds, the gateway runs on 127.0.0.1:8787, and your first OpenAI/Anthropic SDK call appears in Captured calls the moment it lands. No code changes — just one env variable.

Step 1 of 3

Install the CLI.

The kolm CLI bundles the gateway, the local compiler, and the device manager. Node 20+ recommended.

npm i -g github:kolm-ai/kolmogorov-stack
kolm login --key YOUR_KS_KEY
kolm doctor
Step 2 of 3

Point your SDK at the gateway.

Set one environment variable. Any OpenAI or Anthropic SDK now routes through Kolm; calls are captured, signed with a receipt, and (where you've allowed it) cached for distillation.

# Start the local gateway (default port 8787)
kolm gateway start

# Then point any SDK at the same base URL
export OPENAI_BASE_URL="http://127.0.0.1:8787/v1"
export ANTHROPIC_BASE_URL="http://127.0.0.1:8787/v1"
Step 3 of 3

Fire one test call and watch it appear.

Any OpenAI SDK call works. The receipt comes back signed; the capture lands in /account/captured within a second.

curl http://127.0.0.1:8787/v1/chat/completions \
  -H "content-type: application/json" \
  -d '{"model":"gpt-4o-mini","messages":[{"role":"user","content":"hello kolm"}]}'
