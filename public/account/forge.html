Open-source AI workbench

Forge.

Turn a base model and a capture stream into a compiled .kolm artifact you can run on any hardware. Inspect what you have, fit it to your VRAM, quantize, merge, serve, export. Every action returns a verifiable receipt.

Tools

01 · detect Hardware Local GPU(s), VRAM, compute capability, native dtypes, supported quants. GET /v1/hardware 02 · compile Quantize Pick a target (gguf-q4km, exl2-6bpw, nvfp4, fp8...) and build a .kolm. POST /v1/quantize 03 · combine Merge Linear, slerp, ties, dare, passthrough. Dry-run prints the plan first. POST /v1/merge 04 · deploy Serve Emit a docker run or Kubernetes manifest for vLLM, llama.cpp, Ollama, TGI, SGLang. POST /v1/serve 05 · measure Bench K-Score + throughput across all quant targets, side-by-side comparison. POST /v1/bench (with --all-targets) 06 · analyze Experts (MoE) Per-expert activation distribution, prune candidates, estimated K-Score impact. POST /v1/experts

What this surface owns

Forge owns everything from base-model selection to deployable artifact: inspection, memory-fit planning, quantization to 17+ formats, expert pruning for MoE models, model merging, runtime export, and deployment-manifest emission. Backed by the forge-* modules in src/; same source-of-truth that drives the kolm CLI verbs.
