kolm on Azure MarketplaceListing prep

Frontier AI on your own infrastructure. The Azure equivalent of the AWS stack: a Resource Group containing a Virtual Machine running the kolm gateway, a Key Vault for secrets, a Storage Account with a Blob container for signed .kolm artifacts, and Application Insights for audit plus operator telemetry. What you get (planned)

  • Resource Group isolation. Every resource the template creates -- VM, NIC, NSG, vNet, Storage Account, Key Vault, App Insights -- lives in a single Resource Group you name at deploy time. Tear-down is one az group delete call.
  • Azure Key Vault for secrets. Your kolm API key and optional teacher key live as Key Vault secrets. The VM authenticates via Managed Identity; no secrets ever land in template parameters or VM environment.
  • Blob Storage for artifacts. A private Storage Account with a single container named kolm-artifacts. Public access disabled at both the account and container level, AES-256 server-side encryption, soft-delete on for 30 days.
  • Application Insights integration. Audit and operator log streams forward to App Insights so existing Azure Monitor dashboards and alert rules pick them up natively.
  • Network Security Group lockdown. Inbound rules limit port 22 to your operator CIDR and port 443 to your client CIDR. Default-deny on everything else.

Pricing on Azure (estimated)

Bring your own kolm Pro license. Azure bills you only for the VM, Storage, Key Vault, and App Insights resources this template creates. Estimated cost for a typical workload sits between $140 and $320 per month pay-as-you-go:

  • Standard_D8s_v5 in East US: ~$260/month
  • 128 GB premium SSD: ~$18/month
  • Blob Storage standard (50 GB typical): ~$1/month
  • Application Insights ingest + retention: ~$5-20/month depending on traffic
  • Key Vault: pennies/month

Reserved VM Instances cut the compute line by roughly 35-50%. NC-series VMs land here for GPU distill workloads at higher cost.

Deployment (planned)

Once the bicep template lands, deployment will look like this:

$ az deployment group create \
    --resource-group kolm-prod \
    --template-file infra/azure-marketplace/main.bicep \
    --parameters \
      kolmApiKey=ks_xxxxx \
      namespace=prod \
      location=eastus

# Grab the gateway URL
$ az deployment group show \
    --resource-group kolm-prod \
    --name main \
    --query 'properties.outputs.gatewayUrl.value' -o tsv

The bicep template will output gatewayUrl, artifactContainer, keyVaultName, and appInsightsConnectionString so downstream IaC can consume them.

For reference shape, see infra/aws-marketplace/cloudformation.yaml in the repository.
