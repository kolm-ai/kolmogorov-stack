How it works

This is the first distilled_model-class artifact in the seed catalog. The recipe carries a TF-IDF-style keyword model that mirrors the behavior a student model distilled from the base would learn for this routing task. The artifact bundles its own model.gguf weights blob inside the .kolm zip (8 KB deterministic byte stream produced by the build script's sha256 chain), so the validator at recipe-class.js:247 sees a real manifest.hashes.model_weights entry rather than a pointer doc. The evaluation block is grounded by prepareSeedSplit() against 60 real customer-support utterances, yielding 48 train / 12 holdout rows with eval_provenance: real_eval and eval_source: tenant_captured. The recipe response includes a source field so each row's expected output is unique, defeating the train/holdout leakage detector at productionReady.js.

Intents covered

  • refund, refund, money back, chargeback, reimburse.
  • cancel, cancel subscription, cancel my plan, terminate account.
  • billing, invoice, charged twice, billing error, payment.
  • shipping, where is my order, tracking, delivery, shipped.
  • password_reset, reset password, forgot password, can't log in.
  • account_lock, account locked, too many attempts, 2fa.
  • complaint, terrible, angry, unhappy, worst experience.
  • feedback, great service, love your app, suggestion.
  • escalate, speak to manager, supervisor, urgent.
  • other, default fallthrough when no class scores above threshold.

Why this artifact exists

The kolm marketplace previously shipped five rule-class artifacts and one synthesized-rule artifact, but zero distilled_model entries. The audit on 2026-05-20 flagged the absence of a real model-class seed as the single biggest gap: customers who came in via the kolm distill path had no in-marketplace reference for what a production-ready distilled artifact looks like. closes that gap. The build script lives at scripts/build-distilled-model-seed.mjs and is deterministic across runs (sha256 chain for the synthetic weight blob), so the manifest sha256 anchor is reproducible.
