WF01 · Design system v1

A token-driven system, built to the same bar as Linear and Stripe.

Every token below resolves through CSS custom properties so a single swap on [data-theme] retunes the entire surface. No raw hex values appear in component CSS — that is the WF01-5 contract. Components ship dark mode first; light mode is the override.
