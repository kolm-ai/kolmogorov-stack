Golden Path tutorials.

Six end-to-end walkthroughs. Each ends with a signed .kolm artifact in your hands. Each tutorial publishes a verifiable receipt so the tutorial is its own proof.

01

Compile your first PHI redactor in 12 minutes.

Take a redaction task description, hit one command, get a 9.4 KB signed artifact you can drop into a hospital's environment with a HIPAA-shaped receipt.

12 min/healthcare 02

Drop-in replace OpenAI in your codebase.

Point chat.completions at a local kolm endpoint. Same request shape, same response shape, 11.6× cheaper, runs offline. We measure the delta as you go.

14 min/openai-compat 03

Sign and verify in CI.

Set up GitHub Actions to compile, sign, and verify every model artifact your team produces. Verifies the binder. Posts the receipt as a comment on the PR.

11 min/ci-cd 04

Triage PagerDuty alerts in 15 minutes.

Compile a 4.8 KB .kolm that classifies incoming alerts into severity and routing team. Base Qwen 2.5 3B. K-target 0.88. Runs on a single CPU; alert text never leaves your VPC.

15 min/devops 05

Extract clauses from a Master Services Agreement.

Compile a 5.4 KB .kolm that pulls arbitration, IP, payment, and non-compete clauses with character spans. Base Mistral 7B. K-target 0.91.

20 min/legal 06

Compile a PR review bot.

Three issue classes only: missing tests, unbounded recursion, hardcoded secrets. Base DeepSeek-Coder 6.7B. K-target 0.87. Posts inline review comments on GitHub PRs.
