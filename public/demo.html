The 3-minute kolm demo

Capture 20 prod calls, distill to a 1.5B student, compile to GGUF, deploy to a Jetson, verify the receipt. End to end.

What happens in the video

You watch one indie developer take a working OpenAI app, route 20 calls through the kolm gateway to capture them, distill a 1.5B student from that capture pool, compile it to a Q4_K_M GGUF, deploy it to a Jetson Orin Nano on their desk, and verify the runtime receipt — in three minutes. No cuts, no narration, no slides.

  1. Install the CLI + sign up npm i -g github:kolm-ai/kolmogorov-stack && kolm signup --email you@example.com
    0:00 — 0:18
  2. Create a namespace + route the existing OpenAI calls through the kolm gateway kolm namespace create my-skill --primary-provider openai && export OPENAI_BASE_URL=https://api.kolm.ai/v1
    0:18 — 0:42
  3. Send 20 calls (your existing app code, unchanged) for i in {1..20}; do node my-existing-app.js; done
    0:42 — 1:05
  4. Approve the captured pool kolm captures approve --namespace my-skill --status pending --all
    1:05 — 1:18
  5. Compile to a 1.5B student, GGUF Q4_K_M kolm compile --namespace my-skill --target gguf-q4km --base-model Qwen2.5-1.5B-Instruct
    1:18 — 2:05
  6. Deploy to a Jetson Orin Nano via SSH kolm deploy artifact.kolm --target-profile jetson-orin-nano --device jetson-1 --port 8766
    2:05 — 2:38
  7. Hit the local endpoint + verify the receipt curl http://jetson-1:8766/v1/chat/completions ... && kolm verify $RECEIPT_ID
    2:38 — 3:00

Try it yourself, right now

Every command in the video is real. If you have an OpenAI app today, you can do steps 1-4 in ten minutes without leaving your editor:

npm i -g github:kolm-ai/kolmogorov-stack
kolm signup --email you@example.com
kolm namespace create my-skill --primary-provider openai

# point your existing app at the gateway (1 env var change)
export OPENAI_BASE_URL=https://api.kolm.ai/v1
export OPENAI_API_KEY=ks_...   # the key from `kolm signup`

# run your app normally for a while, then:
kolm captures stats --namespace my-skill
kolm captures approve --namespace my-skill --status pending --all
kolm compile --namespace my-skill --target gguf-q4km

The full indie loop is documented

For the deploy + verify steps in the back half of the video: /docs/indie-loop walks each command with the expected output. Bring your own Jetson, Raspberry Pi 5, or Mac mini and the loop runs end-to-end.
