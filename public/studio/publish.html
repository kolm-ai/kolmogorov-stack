Studio · publish to Hugging Face

Publish your .kolm to Hugging Face.

Auto-generate the README from your passport — K-Score axes, teacher council, hardware footprint, runnable examples for transformers, vLLM, llama.cpp, Ollama, MLX. Push to your org. License + citation block included.

Which artifact?

Pass the artifact ID or path. The publisher reads the passport so the README cites real K-Score numbers, not placeholders.

Where on Hugging Face?

Repo path is <org>/<name>. Defaults to your account org if you leave it blank. The publisher creates the repo if it doesn't exist; existing repos get a new commit.

Which license?

License inheritance matters: a student trained on a teacher's outputs inherits that teacher's terms unless your contract overrides. The publisher emits a license_inheritance note in the model card.

Preview the model card, then push.

Preview is rendered from the passport you'll see at kolm passport <artifact>. Push writes the README, uploads the artifact, sets license + tags.
