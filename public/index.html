01 / Operating loop

Control the loop. Export the proof.

The control center treats AI/API traffic as governed state, not loose logs. Every useful call can become a trace, policy decision, eval row, compile input, signed artifact, receipt, target recommendation or export.

01

Collect every useful signal

Provider routes, SDKs, gateway logs, traces, webhooks, batches, queues, warehouses and agent tools arrive with source, tenant and permission context.

  • 17 channel families with opaque-event fallback
  • 12 ingress modes across API, files and data planes
  • Provider vault, source controls and capture policy
02

Gate changes before they ship

Policy, redaction, retention, labels, eval suites, failure clusters and regression sets decide whether behavior is ready to compile.

  • 8 governance stages visible in the console
  • Pioneer-style improvement loop without benchmark overclaiming
  • Readiness-gated claims tied to product evidence
03

Compile artifacts and receipts

A .kolm artifact packages manifests, executable recipes, evaluators, examples, hashes and receipts so proof survives outside the dashboard.

  • Runtime target matrix for local, hosted and restricted fleets
  • 10 export modes for governance, SIEM, catalogs and GRC
  • Verifier bundles for auditors and platform teams

03 / Runtime targets

Shrink to the smallest place the workload can live.

Kolm does not force every call back through the biggest frontier endpoint. It keeps the training and routing evidence with the artifact, then aims the resulting behavior at a concrete runtime target.

Phone Export path and fit checks for tiny local tasks.
Laptop Local CPU, CUDA, DirectML, MLX and MPS adapters.
Server vLLM, SGLang, TGI, TRT-LLM and SSH paths.
Edge ONNX, WASM and constrained runtime probes.
Hosted RunPod, Modal, Vast, Lambda and brokered compute.

Scope note: current artifacts package executable recipes, examples, evaluators, tokenizer metadata and receipts. Full trained-weight bundling and mobile runtime parity remain gated work in the product roadmap.
