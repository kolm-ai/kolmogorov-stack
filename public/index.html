Capture

Route your model traffic through one endpoint.

One endpoint governs every prompt, completion, tool call and routing decision — each one redacted, replayable, and exportable, with no SDK rewrite.

  • Drop-in proxy · keep your provider keys in one vault
  • Redaction before persistence · secrets never stored
  • Export to JSONL, OTLP, or a signed receipt

Compile

Turn captured behavior into a signed artifact.

Kolm packages your examples, recipes, evaluators, tokenizer metadata and receipts into one portable .kolm — signed with Ed25519, reproducible, and inspectable layer by layer.

  • Deterministic build · content-addressed receipts
  • Evals and protected-slice gates before promotion
  • Open the artifact — every layer is governed state

Deploy

Run it on the smallest runtime that fits.

Kolm ranks every target — hosted, server, laptop, edge, phone — by cost, latency and fit, then installs and serves the same artifact wherever you choose. Your data never leaves your hardware.

  • Device detection · quantization oracle
  • One artifact, many targets · hosted or self-managed
  • Rollback, receipts and governance exports included
