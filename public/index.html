Frequently asked

What the compiler actually owns.

One product should own the path from real API behavior to governed, portable runtime artifacts — here is what that means in practice.

Do I have to rewrite my SDK calls?

No. Point your existing OpenAI- or Anthropic-compatible calls at the Kolm endpoint. The gateway captures, redacts, and routes without an SDK rewrite.

Where do my secrets live?

Redaction happens before persistence, so secrets are never stored. Provider keys stay in one vault and never enter the captured record.

What is inside a .kolm artifact?

Examples, recipes, evaluators, tokenizer metadata and receipts — signed with Ed25519, reproducible, and inspectable layer by layer.

Where can I run the artifact?

Anywhere it fits. Kolm ranks hosted, server, laptop, edge and phone by cost, latency and fit, then serves the same artifact on the target you choose.
