What program offices tell us.

Concrete pains we hear from prime integrators and government program offices.

CUI and ITAR exposure

  • SaaS LLMs are not authorized for CUI handling under DFARS 7012
  • ITAR export controls forbid foreign-national operator access
  • FedRAMP High is not enough for many programs

Air-gap constraint

  • SCIFs and forward sites have no inbound internet
  • Model updates must move over sneakernet with signed bundles
  • Hosted inference is not an option at all

Mission-specific reasoning

  • ISR triage, ATO chat, after-action analysis differ per program
  • Generic models hallucinate doctrine and platform specs
  • On-prem retraining with cleared data is required

Architecture: zero cloud egress.

Capture, compile, and inference all run inside the SCIF. The only thing kolm ships is signed software bundles over sneakernet.
