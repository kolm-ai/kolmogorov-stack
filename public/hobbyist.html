Five-command first model

From install to a model you own.

Paste this into a terminal on the box you already use. The install script is a single shell file; read it before piping if that's your habit.

terminal · bash Copy & run 
# 1. Install the kolm CLI from GitHub. Needs Node 18+.
$ npm i -g github:kolm-ai/kolm

# 2. Create a free account. No card.
$ kolm signup

# 3. Quantize a 7B model to 4-bit on your local GPU.
$ kolm quantize Qwen/Qwen2.5-7B-Instruct --bits 4

# 4. Replay a JSONL of example traffic to build a capture corpus.
$ kolm capture replay sample.jsonl

# 5. Distill with Cerebras as the free teacher.
$ kolm distill --teacher cerebras:llama3.1-8b

# 6. Ship to a local Ollama runtime.
$ kolm deploy --target ollama
What is not free

The boundary, stated plainly.

If you outgrow the hobbyist tier, here is what crosses the paid line. Everything else stays free.

  • SSO / SAML — identity-provider integration, SCIM provisioning.
  • BAA — HIPAA business-associate agreement and the controls behind it.
  • Multi-tenant orgs over 5 members — team plan extends seats and role-based access.
  • Premium support — named contact, response SLAs, off-hours coverage.
  • Dedicated training cluster — reserved GPU pool for repeatable distill runs.
  • Air-gap / on-prem deploy — private registry, signed-artifact rollouts behind your firewall.
