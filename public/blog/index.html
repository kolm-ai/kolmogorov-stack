Cadence

Three rhythms. One discipline.

The cadence is fixed so writers, readers, and reviewers all know what to expect. Each format has a different scope, a different fact-base, and a different review path.

Cadence Format Scope Fact-base
Weekly Recipe, walkthrough, or short essay One concrete technique, one model class, one shipped artifact Reproducible from a CLI command and a public dataset
Monthly State of Distillation report What shipped from kolm and from the community in the prior month Aggregated production telemetry plus public artifact registry
Annual Cost of AI roundup What the year of frontier API spend, distillation savings, and on-device deploys actually cost Multi-tenant billing aggregates plus public benchmark history
