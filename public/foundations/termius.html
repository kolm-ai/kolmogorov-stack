Termius recipe

Termius is the SSH client that survives a phone restart, switches networks gracefully, and keeps your keys in the Secure Enclave. Pair it with kolm and you can tail a production capture stream from a coffee shop, kick off a distill from the train, or check why a compile stalled while you walk to lunch.

Why pair them

The kolm CLI is text in, text out, line-buffered. Every verb works under any TTY. That is the property a mobile shell needs. Termius takes your existing SSH key, keeps it in the Secure Enclave or Keystore, and gives you a real terminal on the phone. The combination removes "I am away from my desk" as a reason to wait on a compile or miss a capture spike.

Nothing in kolm assumes a desktop terminal width. Tables truncate cleanly at 60 columns. kolm tail captures prints one row per line. kolm jobs list fits a narrow screen. kolm watch <id> follows a log file just like tail -F does.

One-time setup

  1. Install Termius (termius.com) on your phone. Sign in or create an offline keychain.
  2. Generate an SSH key on the device (Settings → Keychain → New Key → Ed25519). Copy the public key.
  3. Authorize the public key on each box where you run kolm: echo "<key>" >> ~/.ssh/authorized_keys.
  4. Add each box as a Host in Termius. Set the alias to the same string you use on your laptop so muscle memory carries.
  5. In Termius Snippets, save the four verbs you will use most: kolm jobs list, kolm tail captures, kolm watch, kolm distill --from-captures. One-tap to send.

The daily flow

# from your phone, after SSH-ing into the compile box

# is anything running right now
kolm jobs list

# what just happened on the production namespace
kolm tail captures --namespace prod --limit 20

# follow a long compile while on the move
kolm watch job-abc123def456

# the capture stream said the namespace just crossed 1000 pairs
# kick off a distill before you forget
kolm distill --from-captures prod --min-pairs 1000

What this gets you

  • Operational awareness without opening a laptop. You can answer "what is the GPU doing right now" in 6 seconds from anywhere with cell service.
  • Capture-spike response time drops from "next time I am at my desk" to "right now from my phone."
  • Termius Sync (optional, paid) keeps your hosts, keys, and snippets in sync across iOS, Android, and desktop. If you do not want the cloud sync, the offline keychain works.
  • Tied to tmux: SSH into a tmux session from Termius, detach when you put the phone down, reattach from your laptop later. The session does not care which device drove it.
Termius is not a kolm dependency. Any SSH client works: OpenSSH, PuTTY, iSH, ConnectBot, Blink. We name Termius because it is the one most fleet operators standardize on and the only one with a Snippets feature designed for this kind of one-tap recall.

Failure modes worth knowing

Cell network jitter. A flaky cell connection drops the SSH socket. tmux handles this on the server side; without tmux you lose the foreground process. The fix is always tmux new -s kolm or tmux attach before you run anything you care about.

Keyboard ergonomics. Long flags hurt on a phone keyboard. Use Termius Snippets for the verbs you call most. kolm profile use mobile can also set an env profile that sets sensible mobile defaults (smaller --limit, less verbose output).

Push notifications. Termius does not show OS notifications when output arrives. For real alerting, wire the W215 push handler in kolm to your phone instead of trying to watch the screen.
