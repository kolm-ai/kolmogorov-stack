kolm.ai for Claude Desktop

One command wires kolm into Claude Desktop as an MCP server. The installer detects your platform and merges into the right claude_desktop_config.json on macOS, Windows, or Linux without overwriting other servers.

Install command

copy · paste · run
node scripts/install-mcp.cjs claude-desktop

Already have kolm installed globally? Run kolm install claude-desktop instead. Pass --dry-run to preview the JSON without touching disk.

target config: platform-detected (see paths below)

What this does

The installer reads claude_desktop_config.json at the platform-correct location, parses the JSON, merges in a single mcpServers.kolm entry that points to kolm serve --mcp, and writes atomically using a tmp-file + rename. Other servers in the same file are preserved byte-for-byte. Re-running is a no-op when the kolm block is already correct.

The exact block injected:

{
 "mcpServers": {
 "kolm": {
 "command": "kolm",
 "args": ["serve", "--mcp"],
 "env": {}
 }
 }
}

If an existing mcpServers.kolm block differs from the canonical one, the installer prints a diff and refuses to overwrite. Pass --force to replace it.

After install

step 1

Quit and reopen Claude Desktop

Claude Desktop only loads MCP servers at startup. A full quit-and-reopen picks up the new server.

step 2

See kolm tools appear

Open the tools menu in the chat composer. Each .kolm artifact you have compiled appears as a callable tool.

step 3

Call from chat

Ask Claude to use a compiled task. It routes to the local kolm server, signs the call, writes a receipt.

Cross-platform paths

  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: %APPDATA%\Claude\claude_desktop_config.json
  • Linux: ~/.config/Claude/claude_desktop_config.json

Troubleshooting

If the installer prints "refusing to overwrite", you already have a different mcpServers.kolm entry. Inspect the diff, then either edit by hand or rerun with --force. The installer never crashes on an unknown platform: it prints the JSON block and the best-guess path so you can paste manually.
