CLI commands

Grouped by the job: capture, build, run, verify, operate.

capture · ingest production calls

kolm captureDrop-in proxy for OpenAI and Anthropic. Captures (input, output) into a namespace. kolm tailStream live capture rows from your kolm cloud over Server-Sent Events. kolm labelsDownload a captured namespace corpus as JSONL or JSON. kolm seedsScaffold and manage seeds.jsonl training corpora with curated templates.

build · compile, distill, train

kolm compileBuild a .kolm artifact from a cloud-synthesised task or a local JSON spec. kolm distillPromote a reviewed namespace into a local student candidate via the trainer bridge. kolm tuneEvolve a local adapter or rule artifact from skeleton to a promoted revision. kolm pipelineEnd-to-end compile pipeline. Tokenize, distill, compile, full. kolm quantizeQuantize a model checkpoint to int4 / int8 / GPTQ / AWQ. kolm spec-decodeTrain EAGLE-style speculative-decoding pairs through an external trainer bridge. kolm nlNatural-language recipe scaffolder. Plain English in, kolm spec out. kolm init-agentScript-first project scaffolder. Templates for chatbot, redactor, classifier.

run · execute and serve

kolm runExecute a .kolm artifact locally against a JSON or string input. kolm serveExpose .kolm artifacts as MCP tools or as an OpenAI-compatible HTTP server. kolm replayReplay captured prompts against a compiled artifact and report per-row diffs. kolm runtimePer-call policy decider plus runtime-target packaging for cross-compile. kolm ragAirgapped local retrieval (BM25, no embedder, no network). kolm demoCanonical close-the-loop demo. Seeds the lake with ~150 example pairs.

evaluate · bench, gate, score

kolm evalRe-run a .kolm's embedded eval set and show per-case pass / fail. kolm benchReproducible benchmark for a .kolm artifact or a public reproducer suite. kolm loopRun the value-loop smoke. Five rungs, green or red. kolm fixAuto-iterate on a failing artifact. Show failing cases and suggest seed fixes.

inspect · read-only sanity

kolm listShow every local .kolm artifact. kolm inspectShow what a .kolm artifact is in plain text. Manifest, recipes, signature. kolm inspectionThe five read-only verbs you reach for when something is wrong. kolm explainPlain-English description of a .kolm artifact. kolm artifactsList, show, and diff compiled .kolm artifacts in your tenant. kolm logsTail the local run-history log. kolm statusOne-line snapshot of where you are. No network. kolm whatOne-screen snapshot of where you are. Artifacts, captures, jobs, next steps. kolm nextRecommended next action. 1-3 high-value verbs with copy-paste commands. kolm doNatural-language intent dispatcher. Plain English in, kolm verb out.

ops · jobs, sessions, mesh

kolm jobsList, show, and prune the local job registry. kolm watchTail a job log file with rotation-safe follow semantics. kolm sessionsList, inspect, kill, or prune detached worker sessions. kolm resumeRe-attach to a detached compile, distill, eval, or serve session. kolm rescueRe-parent an orphaned kolm child PID under a new controlling tty. kolm agentBuild, inspect, merge, and export self-growing Hermes-style tool-use agents. kolm meshDiscover Tailscale-joined kolm nodes and plan cluster assignment. kolm teamMulti-tenant workspace management. Create teams, invite members, manage roles. kolm profileMulti-tenant config switcher. Save and activate per-tenant kolm settings. kolm cloudReal GPU fine-tunes and bring-your-own-cloud deploys. kolm syncCloud-sync state and legacy git-mirror push / pull. kolm keysSigning-key rotation lifecycle. Generates Ed25519 keypairs. kolm modelsInspect the frontier catalog and pre-cache model weights from HuggingFace. kolm doctorSanity-check the environment. Detect GPU. Run the value-loop smoke. kolm healthPing the configured cloud base and report round-trip timing.

Core HTTP routes

MethodPathPurposeAuth
POST/v1/compileCloud compile of a task into a signed .kolm artifact.Bearer
GET/v1/compile/:idRead a compile job. Returns status, K-score, artifact CID.Bearer
POST/v1/runExecute a recipe or artifact against an input. Returns a signed receipt.Bearer
POST/v1/verifyRe-run the 7-check audit on an uploaded .kolm.Bearer
GET/v1/receipts/:hash/publicRead a public Ed25519-signed receipt chain by hash.Public
GET/v1/capture/streamSSE live tail of new captures (auth + tenant + namespace scoped).Bearer
POST/v1/distill/from-capturesPromote N captured pairs into a recipe or specialist artifact.Bearer
POST/v1/replayReplay captured prompts against a fresh artifact. Diff K + latency + cost.Bearer
GET/v1/artifactsList artifacts. Filter by tenant, namespace, status.Bearer
GET/healthLiveness probe. 200 OK with build slug. Public.Public

Full route catalog at /docs/api and /openapi.json. Every route is rate-limited per tenant and returns receipt-style envelopes.

CLI exit codes

CodeMeaningTypical trigger
0SuccessVerb completed cleanly.
1Generic failureHTTP non-2xx, unmet gate, verification mismatch.
2Bad arguments / unreadable inputMissing required flag, malformed .kolm file.
3Auth missing or rejectedKOLM_KEY empty in CI, expired bearer.
4Not foundArtifact id, namespace, or job id absent.
5ConflictConcurrent compile, registry name collision.
6Quota exceededPlan cap on compiles, captures, or seats.
7Capture store unavailableServer returned 503 on a capture write.

Receipt + durability response headers

HeaderMeaning
x-kolm-capture-idStable id assigned to the capture row. Use this when filing a /v1/replay request.
x-kolm-capture-durabletrue if the row was committed to the configured durable store. Missing means the write did not persist and the client should retry.
x-kolm-capture-demotrue when the response came from an anonymous demo endpoint. Receipt body carries demo:true.
x-kolm-distill-readytrue once the namespace crosses 100 / 500 / 1,000 pairs (W215). Triggers downstream alerts.
x-kolm-distill-thresholdThe threshold tier just crossed. One of 100, 500, 1000.
x-kolm-buildBuild slug. Matches the sw.js cache slug. Useful for client-side staleness checks.
x-kolm-tenantEcho of the resolved tenant id from the bearer token. For audit.

.kolm artifact layout

Path inside .kolmPurpose
manifest.jsonTop-level manifest. Carries CID, K-score, gate, provenance, frozen-eval pointer.
manifest.sigEd25519 signature over manifest.json. Required.
recipes/*.jsonDeclarative recipes. Each is its own signed unit.
seeds.jsonlSeed pairs used to compile. Reproducibility input.
holdout.jsonlHoldout set scored against the gate. Disjoint from seeds.
receipts/*.jsonSignature-linked receipts. Each receipt binds the previous receipt hash.
provenance.jwtActor key fingerprint, build host, CI run id. Verifiable independently.
README.mdHuman-readable summary. Optional but recommended for marketplace listings.

Full file-format spec at /spec/rs-1. The hosted verifier at /verify-prod reads this layout directly.
