2026-05-26

Trinity-500: distilled 7B student published with GGUF, model card, and benchmark

The Trinity-500 council distill (Claude, GPT-4o, DeepSeek-R1 as teachers) shipped end-to-end on an RTX 5090: 410 council pairs, LoRA adapter trained in 79 seconds, and full GGUF quant ladder (Q4_K_M, Q5_K_M, Q8_0, IQ4_XS). The model card is generated from the passport file so provenance is verifiable. Benchmark (n=57): trinity-500 reaches 96.5% asks-1Q, 100% judge-clarify, and 100% judge-on-policy at 1.24 s and 210 chars per response — beats claude-haiku-4-5 and ties gpt-4o-mini at half the chars. See /models.

Teacher proxy is now the production default + new kolm teacher verb

Distill workers and the bake-off harness now route through the hosted teacher proxy by default, so the same artifact compiles identically whether you have a frontier provider key locally or not. A new kolm teacher CLI verb prints provider status, where keys come from, and where the compiled bytes land. The dotfile loader reads ~/.kolm/.env so credentials never live in shell history.

The gateway wrapper surface closed out with a Vercel teacher-chat proxy fallback. Post-ship prod benchmark: 10/10 receipts and 10/10 upstream 2xx, mean dispatch latency 2.48 s, $0.307 per 1 000 calls vs. $0.301 direct — a 2% wrapper tax with full capture + signed receipts on every call. See /gateway and verify any receipt at /verify-prod.

Frontier quantization sprint: MoE awareness + full quant ladder + hardware-aware select

SSO / SAML / SCIM behind the enterprise plan + Qwen2.5-32B INT4 verified

Six new SSO routes land behind a plan gate. Wraps the trace capture and the workflow-IR compile into one loop: compile a trace into a replay artifact, then verify that the artifact reproduces every step. Cache-hit replay returns the original output without re-running the LLM or tool steps. Foreign trace IDs fail loud with HTTP 403.

Federated approval-row sharing with Laplace noise

Only the approval hash crosses tenant boundaries: sha256 of namespace plus input hash plus decision kind. Raw input and output text never leave the tenant. Peer counts are differentially private (epsilon = 1.0, Laplace scale = 1.0). Opt-in registry is durable.

2026-04-22

Drift snapshot and verdict bands

Drift snapshots compress the K-score and per-axis composite into a small JSON blob. The drift detect endpoint compares two snapshots and returns a verdict band: within, degraded, broken. The bands gate auto-promotion of new artifacts behind a numeric threshold rather than developer eyeball.

How to read this page

Format and reading guide for engineers, procurement reviewers, and auditors who landed here from a sales deck.

formatStable

Every entry is a shipped change

If a line item appears here, the artifact behind it is reachable from /verify-prod by CID. Plans, roadmap items, and in-flight work do not appear on this page — they live on the roadmap.

formatStable

Dates are calendar UTC

Date pills reflect the day the change merged to main and the receipt was signed. Deploy is usually within an hour; deploy events for incidents are linked from the status page.

scopePlain

Marketing fluff is forbidden in entries

Breakthrough adjectives and unqualified speed claims are banned by a test on this page and every other public surface. If a release notes line sounds like it came from a sales deck, file a bug.

linksStable

Every link points to a real surface

The audit-href and audit-static-refs scripts run on every commit. If a link on this page ever 404s, it failed CI before it shipped. Report any broken link to trust@kolm.ai.
