3

Run it anywhere

Choose where your model lives. Same model, different home.

Local means the model runs on the computer in front of you. Nothing leaves your device — best for privacy, and it is free. You need a recent Mac, Windows, or Linux machine.

Get the desktop app How local works
▸ Advanced: one-liner
Run your model locally
kolm run my-model --local

Cloud means we rent a GPU for you and host the model. No hardware required and it is reachable from anywhere by a link. You pay only for the time it runs.

Deploy to the cloud Pick a GPU
▸ Advanced: one-liner
Serve your model on a rented GPU
kolm serve my-model --cloud

Exported means you take the model somewhere else. Hand it to Ollama, publish it on Hugging Face, or just download the file and keep it.
