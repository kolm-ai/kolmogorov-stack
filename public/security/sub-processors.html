Security · sub-processors

Every third party with access to customer data.

This list is current. We notify by email + update this page 30 days before adding a new sub-processor; you have 30 days to object per the DPA. Subscribe to the change feed at the bottom.

Caveats: this list excludes service providers that never touch customer data (e.g. payroll, accounting). Frontier-model providers (OpenAI, Anthropic, Google) are sub-processors only when you route through them via the Wrapper.
Core infrastructure

Cloud + delivery.

These are present in every deployment. You can opt out of AWS by self-hosting; the rest run the public service.

AWS
Compute + storage · us-east-1, eu-west-1

S3 (capture lake + artifact store), RDS Postgres (control plane). SOC 2 Type II, ISO 27001, GDPR-compliant. EU mirror is in eu-west-1 only.

Vercel
CDN + frontend hosting · global edge

Serves kolm.ai static pages + /account/*. No customer data persisted on Vercel — control-plane data lives in AWS. SOC 2 Type II.

Railway
API hosting · us-west, eu-west on request

Runs the /v1/* API. Receives in-flight requests; persists nothing — all writes go to AWS S3 + RDS. SOC 2 Type II.

Cloudflare
WAF + DNS + DDoS · global edge

Layer-7 filtering and rate limiting before requests reach origin. Does not see decrypted payloads (mTLS not enforced). SOC 2 Type II, ISO 27001.

Operational

Billing, transactional email, monitoring.

Each receives a narrow slice: Stripe sees billing metadata, Postmark sees email-addressed transactional content, Sentry sees error stack traces with PII redacted before send.

Stripe
Billing · us-west-1

Customer name, email, billing address, payment method. No prompt/completion data. PCI DSS Level 1.

Postmark
Transactional email · us-east-1

Signup, password reset, compile-done, usage alert emails. Sees recipient email + subject + body. SOC 2 Type II.

Sentry
Error monitoring · us-east-1, eu-west-1

Stack traces and breadcrumbs. PII redactor strips request bodies + headers before send. SOC 2 Type II.

HackerOne
Bug bounty intake · us-east-1

Researcher reports, including any reproduction data they submit. SOC 2 Type II.

Conditional

Only present when you opt in.

These appear in your sub-processor list only if you enable the corresponding feature.

Anthropic / OpenAI / Google / Together / Fireworks
Frontier inference · varies by provider

Sub-processor only when you route Gateway calls through them. Listed individually in your namespace's provider block.

RunPod / Modal
GPU compile · varies by region

Sub-processor only when you use kolm compile --cloud runpod or --cloud modal. Receives the captured training data for the duration of the compile job.

Hugging Face
Model card publish · us

Sub-processor only when you use kolm publish --to huggingface. Receives the compiled artifact + model card you author.

GitHub
CI quality gate · us

Sub-processor only when you use kolm-ai/quality-gate-action@v1 in a public repo. Receives the eval set you commit + the K-Score on the PR.
