Compile Qwen2.5 72B to GPTQ on consumer hardware with kolm.

Qwen2.5 72B is Alibaba's 72B Qwen2.5 open-weight instruct model. GPTQ is GPTQ (Frantar et al., 2023) calibration-based 4-bit quantization. Wide vLLM and AutoGPTQ runtime support.

Qwen2.5 72B is an open-weight checkpoint (72B parameters from Alibaba). kolm compile pulls the bf16 weights, runs GPTQ calibration with C4 calibration samples, and emits a signed .kolm artifact with the quantized weights and the original bf16 SHA-256 hashes for reproducibility.

The output is a single GPTQ file plus a receipt. The receipt records every input weight hash, the quantization method, wall-clock duration, and the host hardware profile. You can verify a compiled artifact end-to-end at /verify-prod or with kolm verify ./artifact.kolm.

Resource estimates below are computed from parameter count and target bits-per-weight. They are unverified until you run kolm compile qwen2.5-72b --target gptq on your hardware and a receipt is logged. Caveats: verifying.

One-line command

$ kolm compile qwen2.5-72b --target gptq

Runs the full compile pipeline: download bf16 weights, quantize to GPTQ, sign the receipt, write the .kolm artifact. See the GPTQ compile docs for flags + tuning.

Step-by-step

  1. Install kolm. npm i -g github:kolm-ai/kolm-stack or pip install git+https://github.com/kolm-ai/kolm@main#subdirectory=sdk/python.
  2. Sign up. Create an account and copy your API key. Free tier available.
  3. Run compile. kolm compile qwen2.5-72b --target gptq
  4. Verify the receipt. kolm verify ./artifact.kolm — confirms SHA-256 chain, ed25519 signature, K-score.
  5. Serve or deploy. kolm serve ./artifact.kolm for a local OpenAI-compatible endpoint, or kolm deploy ./artifact.kolm --device my-box to push to a remote machine.

Why kolm

  • Signed receipts. Every compile emits a receipt with input + output SHA-256 hashes, the quantization method, and an ed25519 signature. Verifiable end-to-end at /verify-prod.
  • Reproducible. Run the same compile twice on the same hardware: identical output hashes. The receipt makes it auditable.
  • Runs anywhere. The compiled GPTQ artifact loads in AutoGPTQ / vLLM with no kolm dependency at inference time. You own the weights.
  • Open source. Apache-2.0. GitHub.
