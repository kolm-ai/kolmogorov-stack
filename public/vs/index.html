How to read this matrix

No single tool wins every dimension. Most production teams end up with two or three of these in the stack: a hosted teacher for the long tail, a local runtime for the hot path, kolm to compile the bridge between them.

If you only need to chat with a model locally, Ollama or LM Studio is the right answer. If you need frontier general reasoning on hosted infrastructure, pick OpenAI or Together. If you need a task-specific compiled artifact you own and can ship anywhere, that is what kolm exists for.
