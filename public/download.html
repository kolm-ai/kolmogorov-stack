Download · CLI and desktop wrappers

One CLI. Three native wrappers.

The Kolm CLI is the canonical entry point today. One source install gives macOS, Windows, and Linux the same gateway, compiler, verifier, and runtime.

CLI · Node 20+

The fastest path. Installs the kolm binary globally from the current source package; works on macOS, Windows, and Linux without admin.

npm i -g github:kolm-ai/kolm-stack

source install · verify with kolm --version · then kolm login

CLI docs →Quickstart →

macOS · Apple silicon + Intel

Use the same source install on Apple silicon and Intel Macs. Signed desktop packaging requires release artifacts; the CLI path is the supported install today.

npm i -g github:kolm-ai/kolm-stack
Supported today via CLI

Windows · x64 + ARM

Use the same source install in PowerShell with Node 20+. Signed installer channels require release artifacts; the CLI path is the supported install today.

npm i -g github:kolm-ai/kolm-stack

Linux · source one-liner

Installs the CLI and runtime under ~/.kolm. No root and no systemd unit by default.

npm i -g github:kolm-ai/kolm-stack

source install · /docs/install/linux
